Editable Tree View in Group By

With this module user can edit the attributes available in the tree view (quotation number, order date, customer name, salesperson, and total amount) even when “Group By” is applied.

Prerequisites before Installation

No prerequisites required before installation.

Configurations

No configurations required.

As shown in the figure user can edit any of the attributes displayed in the tree view of Quotations module. And this is applicable to tree-view available on other different modules such as customers, sales order, products, etc. in Odoo.
